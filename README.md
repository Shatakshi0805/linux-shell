# linux-shell

this is a little linux shell implementation in C language. following ![Stephan Brennen's](https://brennan.io/2015/01/16/write-a-shell-in-c/) Write a Shell tutorial.

to use this:
- download the file
- download gcc compiler to compile the C file. command: `sudo apt update && sudo apt install build-essential`
- compile the program: `gcc -Wall lsh.c -o lsh`
- `-Wall` cmd to show all warnings. `-o` to name compiled file `lsh`
- run the executable file: `./lsh`
